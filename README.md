# Inventory Tracker

**Inventory Tracker** is a desktop application built with **Electron** and **Node.js** for managing stock, tracking sales, and analyzing inventory data in an electronic shop.

## Features
- Add items to the inventory and track remaining stock.
- Record sold and faulty/rejected items.
- Update new stock and isolate old stock.
- Export inventory and sales data to Excel for analysis.
- Visualize sales trends and product analytics with graphs and charts.
- Handle discounts and original prices for items.
- Manage inventory efficiently via a user-friendly interface.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Packaging**: Electron
